# SPH1 - Sistema de Manutenção Predial

Sistema completo de manutenção predial com aplicativo mobile para técnicos e painel web para administradores.

## 📋 Visão Geral

O SPH1 permite que:
- **Técnicos** (Mobile) registrem e finalizem chamados de manutenção
- **Administradores** (Web) gerenciem chamados, técnicos e garem relatórios

## 🏢 Estrutura - Condomínio Parque dos Ipês

- **20 Blocos**: 7C, 7B, 7A, 6C, 6B, 6A, 5C, 5B, 5A, 4C, 4B, 4A, 3B, 3A, 2C, 2B, 2A, 1C, 1B, 1A
- **Por Bloco**: 5 andares
- **Por Andar**: 4 apartamentos (01, 02, 03, 04)
- **Total**: 400 apartamentos

## 🛠️ Tech Stack

- **Backend**: Node.js + Express
- **Frontend Web**: React
- **Mobile**: React Native
- **Database**: PostgreSQL
- **Autenticação**: JWT

## 📁 Estrutura do Projeto

```
sph1-manutencao-predial/
├── backend/          # API Node.js
├── frontend/         # Painel Admin (React)
├── mobile/           # App Técnico (React Native)
└── docs/             # Documentação
```
